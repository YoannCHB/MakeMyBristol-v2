Comment ça marche ?


Pour modifier la couleur, dite(avant de dire votre texte): # couleur de votre choix (exemple: # rouge ) Pour stopper la couleur, dite : #stop


Pour ajouter des caractères,dite: le nom du caractère (exemple: point )


Pour surligner le texte, dite: # fluo la couleur de votre choix (exemple: # fluo Jaune )


Pour modifier le titre, dite: # mets titre (...) (exemple: # mets en tritre Maths )


Pour effectuer une recher Wikipédia, dite: # recherche (...) (exemple: # recherche Victor Hugo )


Pour ouvrir un fichier ou une image clicker sur le bouton "ouvrir"


Pour enregistrer votre projet et l envoyer à des amis clicker sur "Enregister"
